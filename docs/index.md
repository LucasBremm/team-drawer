# Documentação do Team Drawer

## Visão Geral

O Team Drawer é uma aplicação web frontend para organização de partidas casuais de futebol com foco em sorteio equilibrado de times.

O produto atende um público misto: grupos de amigos e pessoas externas que queiram organizar jogos casuais.

O objetivo principal do repositório é permitir:

- cadastro local de jogadores;
- seleção dos participantes de uma partida;
- sorteio de dois times com equilíbrio de habilidade e quantidade de jogadores;
- exportação e importação dos dados em JSON;
- compartilhamento rápido do resultado do sorteio via clipboard.
